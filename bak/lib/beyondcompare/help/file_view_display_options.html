File View Display Options

Tools > Options > File Views > Display

These options allow you to customize visual aspects of the various types of file views.

Select an element from the list to change its visual treatment.  You can change the Background color and Text color of each element on the list.  Lower elements in the list override higher ones.  Select Default to leave unchanged a color set by an element higher on the list.

Mark the Use Stripes checkbox to tint the background on every other line of the comparison.

By default, Beyond Compare uses a medium green color for selected text, so that difference coloring is not obscured.  If you prefer the standard Windows coloring for selections, pick Selection from the list of elements and mark Use system.

The list also includes syntax highlighting elements.

Click one of the Font buttons to change the overall font used to display text.  The Editor font is used in Text Compare, Text Merge and Data Compare views.  The Hex font is used in Hex Compare views, as well as the optional hex details display at the bottom of other views.  The Listing font is used in all other views.

The Dim inactive pane by XXX % setting is used to darken the non-active side of the comparison, to help you focus on the active side.
