Exporting Settings

Tools > Export Settings

You can selectively save your program settings, in order to transfer them to another computer, upgrade to a newer version, or share them with another Beyond Compare user.

To export your settings:

1.On the Tools menu, select Export Settings.
2.Select the type of settings you wish to export:

Program Options include your preferences that you set in the Options dialog.

Program Commands include your preferences that you set in the Customize Commands dialogs.

Program State includes things like window positions and MRU lists.

Sessions include some or all of your named sessions.

File Formats include some or all of your customized file formats.

FTP Profiles include some or all of your FTP settings.

System Info includes information about your computer and your Beyond Compare installation.  It is useful for support purposes, to help us debug problems.

3.If you are exporting sessions, file formats, or FTP profiles, click Next to pick the specific named items to be included.
4.Click Finish to export the settings.  A settings package (.bcpkg) file will be created.
