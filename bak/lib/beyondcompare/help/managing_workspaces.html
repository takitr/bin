Managing Workspaces

Session > Load Workspace > Manage Workspaces

A workspace is the collection of all open sessions, including the configuration of windows and tabs.

To save your current workspace configuration:

1.From the Session menu, select Save Workspace As.
2.Enter a unique name or select from the dropdown a saved workspace to be replaced.

To reload a saved workspace:

From the Session menu, select Load Workspace and pick a saved workspace from the submenu.  This will close all current sessions and reopen the saved configuration.

To delete or rename saved workspaces:

1.From the Session menu, select Load Workspace > Manage Workspaces.
2.Select a saved workspace and press F2 to rename it, or DEL to delete it.

To automatically load or save workspaces:

In the Manage Workspaces dialog, you can specify a workspace name in On startup, load and/or in On exit, save as. These options can also be set in Tools > Options > Startup.

You can also load a workspace by specifying it on the command line.
