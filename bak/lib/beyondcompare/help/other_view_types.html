BC2 Plug-ins

Beyond Compare 3 for Windows can use plug-ins designed for version 2 (although most of our version 2 plug-in functionality is now built into version 3).  To install a plug-in, copy the .bcp file to Beyond Compare 3's program folder.  Restart Beyond Compare, select Tools > File Formats , click New and add an appropriate format definition for the plug-in.
