Importing Settings

Tools > Import Settings

You can import settings that were previously exported from BC3 or BC2.

To import settings:

1.On the Tools menu, select Import Settings.
2.Select the file containing the settings to import.  It can be a BC3 Settings package file (.bcpkg) or XML file.  It can also be a BC2 export file (zip or XML file).
3.If the file contains different kinds of settings, mark the checkboxes for the type of settings you wish to import:

Program Options include your preferences that you set in the Options dialog.

Program Commands include your preferences that you set in the Customize Commands dialogs.

Program State includes things like window positions and MRU lists.

Sessions include named sessions.

File Formats include customized file formats.

FTP Profiles include FTP settings.

4.If you are importing sessions, file formats, or FTP profiles, click Next to pick the specific named items to be included.

Warning  An item is colored red if its name is the same as one in your stored settings.  If you import a red item, it will replace your existing session, file format, or FTP profile!

Mark the Delete all existing sessions checkbox to erase all of your stored sessions before importing those in the file.  Likewise, mark Delete all existing file formats or Delete all existing FTP profiles if you wish.

5.Click Finish to import the settings.
