Find

To locate specific text in a file view, select Find from the Search menu.

Text to find

Enter the text to search for.

Match character case

Match only text with the same uppercase and lowercase characters.

Whole words only

Match on the whole words but not parts of words.

Regular expression

Indicates the search string is a PCRE regular expression.  You can use the dropdown button to the right of the search string to insert common regular expression elements.  Click the RE Examples button for more info.

Wrap search

When one end of the file is reached, continue searching at the other end.

Search up

Searches backwards, towards the top of the comparison.

Sides

Specify the side(s) of the comparison to search.

Close dialog automatically

Close the dialog immediately after clicking the Find button.  Use the Search menu's Find Next and Find Previous commands for finding additional occurrences of the search string.
